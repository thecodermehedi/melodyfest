###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>In the given code `sum()` returns the sum of the given two parameters a and b. But when it was given number (`a=1`) and string (`b="2"`) as arguments, javascript performed type conversion and treats the number (`1`) as string (`"1"`). thats why as a result of concatenation of (`"1" + "2"`) was "12".</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>In the given code, the `sayHi()` has a **parameter** `name` to be passed when it is called. Since there is **no argument** provided when it was called the value of name becomes **undefined**.</i>
</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>In the given code, `count` variable is incremented by 1 for each non-zero number of `nums` array. because `0` is **falsy** value, the result will be **3**.</i>

</p>
</details>
